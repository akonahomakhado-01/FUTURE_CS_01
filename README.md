README – Vulnerability Assessment Report

**Websites Tested**
- Target 1: itsecgames.com – bWAPP: A buggy web application used for training and security research.
- Target 2: webscantest.com – Web Scan Test Site designed for controlled vulnerability analysis.

**Scope of Assessment**
The assessment was conducted under strict ethical guidelines and academic boundaries.
Activities included:
- Passive Reconnaissance: Collecting publicly available information (headers, DNS records, technology stack).
- Active Analysis: Controlled port scanning to identify open services and software versions.
- Vulnerability Identification: Detecting missing security headers, protocol weaknesses, and information leakage.
- Remediation Planning: Providing actionable recommendations aligned with OWASP standards and industry best practices.

**Tools Used**
- SecurityHeaders.com → Analyzed HTTP response headers and identified missing protections.
- BuiltWith → Profiled the technology stack, frameworks, and infrastructure components.
- nslookup (Windows CLI) → Resolved domain names to IP addresses and confirmed hosting details.
- Nmap (via HackerTarget) → Conducted controlled port scans to identify open services and server versions.
